# MongoidView

Helps you make easier models against the aggregation pipeline.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mongoid_view'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mongoid_view
